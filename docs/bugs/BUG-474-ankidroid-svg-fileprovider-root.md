## BUG-474 · AnkiDroid外字SVG制卡FileProvider找不到根
- **报告**：2026-06-30（用户：Android 真机日志实证）
- **真实性**：✅ 真 bug。根因 `hibiki/android/app/src/main/res/xml/provider_paths.xml`（缺 code_cache 根）+ 写入目录 `packages/hibiki_anki/lib/src/anki_models.dart:563-564`（`Directory.systemTemp/anki-media`，Android 上 = `/data/data/<pkg>/code_cache`）。FileProvider 调用点 `hibiki/android/app/src/main/java/app/hibiki/reader/AnkiChannelHandler.java:240`（`getUriForFile`）。
- **[x] ① 已修复** — commit 见末行。AnkiDroid 制卡走 `addFileToMedia` → `FileProvider.getUriForFile(…, ".provider", file)`，file 路径是 `…/code_cache/anki-media/hibiki_dict_*.svg`。AndroidX core **1.13.1** 的 FileProvider 只认 `root-path/files-path/cache-path/external*` 标签（实测 javap 反编译 FileProvider.class 确认，**无 `code-cache-path`**：`cache-path`→`getCacheDir()`=`cache/`，与 `code_cache/` 是不同目录），故 provider_paths.xml 没有任何根覆盖 `code_cache` → `getUriForFile` 抛 `IllegalArgumentException: Failed to find configured root that contains …code_cache…`，SVG 外字图附不上卡片。**根因修复选 (B)**（FileProvider 配置补根，非改写入目录）：写入路径 `ankiDictionaryMediaCacheDirPath()` 是**同步** API，被 writer（`dictionary_webview_media.dart`）/ AnkiConnect reader / 多处共用；改 `getTemporaryDirectory()`（async）要改 hibiki_anki 公开 API + 加 path_provider 依赖，跨包 API 变更、blast radius 大。code_cache 本身是合法 app-private 目录，缺的只是 FileProvider 白名单。故在 `provider_paths.xml` 加 `<files-path name="code_cache" path="../code_cache"/>`：FileProvider `addRoot` 对 `getFilesDir()/../code_cache` 取 `getCanonicalFile()` → 规范化成 `/data/data/<pkg>/code_cache`，**精确覆盖该根**（不用宽泛的 `<root-path>` 暴露整个设备根）。同 provider 也被 `${applicationId}.provider` 与 InAppWebView provider 引用同一 xml，reader/video 共用查词制卡链路都受益。
- **[x] ② 已加自动化测试** — `hibiki/test/android/anki_fileprovider_codecache_guard_test.dart`：源码扫描守卫，断言 (1) provider_paths.xml 存在；(2) `ankiDictionaryMediaCacheDirPath()` 仍以 `anki-media` 结尾（钉死「写到 systemTemp/code_cache」前提）；(3) provider_paths.xml 声明了能覆盖 code_cache 的根（`<files-path path="../code_cache">` 或退而 `<root-path>`）。真机 FileProvider 原生解析跑不了，故守配置机制：根被移除或写入目录脱离 code_cache 假设即转红。
- **备注**：与 [BUG-118](BUG-118-anki-gaiji-card-image-missing.md)（AnkiConnect 嵌套 `<img>` 标签坏图）**不同根因**：118 是 tag 契约不对称，本 bug 是 AnkiDroid FileProvider 路径白名单缺 code_cache。androidx core 1.13.1 FileProvider 标签集经反编译核实。
