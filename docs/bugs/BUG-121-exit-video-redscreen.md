## BUG-121 · 退出视频闪红屏（deactivate 期根 Overlay 浮层重建做失效祖先查找）
- **报告**：2026-06-08（用户：「现在退出视频会……闪一下红屏报错」，附 FlutterError 栈：`building LayoutBuilder` → `Looking up a deactivated widget's ancestor is unsafe` → `ProviderScope.containerOf` → `_VideoHibikiPageState.appModel (video_hibiki_page.dart:235)` ← `mixinAppModel:187` ← `DictionaryPageMixin.buildNestedPopupLayer:216` ← `_buildPopupOverlay` 的 LayoutBuilder）。
- **真实性**：✅ **真 bug（运行时栈 + 代码路径确认）**。查词浮层 entry 插在**根 Overlay**（跨路由生存，比本页 State 活得久，`_syncPopupOverlay`）。退出视频/退全屏当帧本 State 先 `deactivate`，但**同帧 layout 阶段**根 Overlay 的 `LayoutBuilder`（`_buildPopupOverlay` 内）仍会重建 entry —— `LayoutBuilder.builder` 在 layout 阶段运行，晚于 deactivate。此刻内层经 `appModel`（`ref.read(appProvider)` → `ProviderScope.containerOf` 做 InheritedWidget 祖先查找）与 `mixinTheme`（`Theme.of(context)`）访问 State 的 `context`/`ref`，而 **deactivated（未 unmount）element 上的祖先查找不安全** → 抛 `FlutterError` → ErrorWidget 红屏。注意 `_buildPopupOverlay` 外层早有 `if (!mounted)` 守卫，但 `deactivate` 期 `mounted` 仍为 true（element 未 unmount），且崩溃在**内层 LayoutBuilder 的 layout 阶段回调**，外层守卫拦不到。`dispose()` 里摘除 entry 太晚（`OverlayEntry.remove()` 在 build/layout 阶段会延迟到 post-frame，拦不了本帧 layout）。
- **[x] ① 已修复** — 本提交。根因修（按生命周期窗口拦截，非吞异常）：新增 `bool _overlayInert`，`deactivate()` 置 true（`deactivate` 在 reconcile 阶段运行，**早于**同帧 layout）、`activate()` 复位（GlobalKey 重挂场景恢复，下次 build 的 `_syncPopupOverlay` 重建浮层）。`_buildPopupOverlay` 外层与**内层 LayoutBuilder builder 起始**都加 `if (!mounted || _overlayInert) return const SizedBox.shrink();`——layout 阶段重建据此空渲染，杜绝对 `appModel`/`mixinTheme` 的失效祖先查找。一处拦截同时覆盖 `appModel`(ref) 与 `mixinTheme`(context) 两类查找。根因 `file:line`：`hibiki/lib/src/pages/implementations/video_hibiki_page.dart`（`_buildPopupOverlay` LayoutBuilder + `deactivate`/`activate`）。
- **[x] ② 已加自动化测试** — `hibiki/test/pages/video_overlay_teardown_guard_static_test.dart`（源码扫描守卫；deactivate↔layout 同帧时序需真 Overlay/路由栈，headless widget 测试难稳定复现）：断言存在 `_overlayInert` 字段、`deactivate()` 置 true、`activate()` 复位、`_buildPopupOverlay` 的 LayoutBuilder builder 在访问 `appModel`/`screen` 之前有 `_overlayInert` 早返回守卫。
