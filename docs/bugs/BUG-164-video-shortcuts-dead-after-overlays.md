## BUG-164 · 视频快捷键失灵：设置/导入/点外部/全屏后空格等失效
- **报告**：2026-06-10（用户：TODO-040/042 群聊提炼——①点外部/切窗返回 ②导入字幕后 ③改设置后 空格等播放快捷键失灵；TODO-017 post-frame 归还焦点后仍复发）
- **真实性**：✅ 真 bug。共同根因（框架级）：media_kit 窗口/全屏两套 controls 共用同一个 `_videoFocusNode`（`material_desktop.dart:581` 经 `videoViewParametersNotifier.value.focusNode`），全屏是独立路由而窗口侧 controls 仍挂在树上 → 同一节点被两个 `Focus` 同时 attach。Flutter `FocusAttachment` 语义（`focus_manager.dart`：`attach` 覆盖 current attachment / `detach` 仅 current 时摘树 / stale `reparent` 永远 no-op / 孤儿节点 `requestFocus` 只置 `_requestFocusWhenReparented` 静默挂起）⇒ **退全屏一次后节点永久孤儿**，此后页面里所有逐点 `_refocusVideo()` 补丁（设置 sheet whenComplete、FilePicker await 返回、导入遮罩 post-frame 等共 15+ 处）全部静默失效——这就是「打过补丁仍复发①②③」的原因。根因位置：`hibiki/lib/src/pages/implementations/video_hibiki_page.dart`（旧 `_pushNeutralizedVideoFullscreen` 全屏路由复用共享节点 + 窗口侧 controls 不卸载）。次级缺口：`_popNestedPopupAt` 关查词浮层栈后无 refocus；点视频画面不回收键盘；窗口重新激活（切窗返回）不回收。
- **[x] ① 已修复** — 统一机制（消特例而非再撒补丁）：新增 `VideoControlsFocusGate`（`hibiki/lib/src/media/video/video_controls_focus_gate.dart`）在全屏路由在栈上时卸载窗口侧 controls，保证任意时刻只有一个 `Focus` 持有共享节点（进/退全屏靠 build-早于-finalize-unmount 的顺序干净交接，节点永不孤儿）；`_videoFullscreenActive` 标记 + 全屏路由 future `.whenComplete(_onVideoFullscreenRouteClosed)` 单点收口（覆盖 Esc/F/按钮/双击/系统返回全部退出路径）；进全屏 `finally` 的 refocus 改 post-frame（同步调用会跑在路由 build 前、被 reparent 冲掉）；`_popNestedPopupAt` 栈空时 refocus（与恢复播放共用唯一关栈汇聚点）；`_handleVideoPointerUp` 点画面回收键盘（原生播放器语义）；`didChangeAppLifecycleState.resumed` 经统一判据 `_reclaimVideoFocusIfOwned()`（页面或全屏路由为当前路由且无查词浮层才回收，不抢对话框焦点）。空格的全局 DoNothingIntent 中和（global_navigation.dart）未动。提交：见本 worktree 分支 `worktree-agent-a2418b4b1d153b928`。
- **[x] ② 已加自动化测试** — `hibiki/test/pages/video_fullscreen_focus_gate_test.dart`（gate 三态单元测试 + 全屏往返机制行为测试：带 gate 空格存活、**无 gate 对照组实证节点孤儿/空格死亡/requestFocus 无效**（若 Flutter/media_kit 语义改变该对照会红、提示重评 gate）+ modal whenComplete refocus 时序假设固化）；`hibiki/test/pages/video_page_keyboard_focus_static_test.dart` 新增 5 条页面接线守卫（gate 接线/全屏单点收口/浮层关栈 refocus/点画面回收/resumed 回收）。真实 `VideoHibikiPage` headless 无 libmpv 不可加载，行为层用与 media_kit `material_desktop.dart` 同构 harness。
