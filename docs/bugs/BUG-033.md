## BUG-033 · 书架右上图标按钮按方向键「下」焦点跳到左侧导航栏（而非下方内容）
- **报告**：2026-06-04（用户：「右上角的统计这三个按钮，按方向键下会去到左边栏，根本性修复类似的问题」）。
- **真实性**：✅ **真 bug（方向焦点「面板」身份用错代理）**。沿真实路径定位：书架 body = `Column[页头(3×HibikiIconButton，无 Scrollable 祖先→`Scrollable.maybeOf`=null) / 标签栏(横向 ListView) / Expanded(CustomScrollView 书格)]`（`reader_hibiki_history_page.dart:141-147,193-214`）；侧栏 rail = `_MaterialNavCluster` 垂直、**非滚动** `Column`（`adaptive_navigation.dart:130-157`）→ rail 项也 null。`HibikiFocusController._geometricTarget`（`hibiki_focus_controller.dart`）的几何排序里 `samePane` 档（tier 1）高于 `along`（纵向距离，tier 2），而 `samePane = identical(Scrollable.maybeOf(target), Scrollable.maybeOf(active))` 把 null==null 视为「同面板」。于是页头按钮(null) 与 rail(null) 被判**同面板**，真正内容（在 Scrollable 里）反被判**异面板**：Down 时 rail 在 samePane 档击败内容；短窗口里 rail 项纵向甚至更近，`along` 也帮 rail。这是「自绘导航簇（桌面 rail / 手机底栏）把每个目的地注册成普通焦点目标却无『列/面板』身份，纵向移动从 chrome/内容误入导航列」这一类问题的统一根因。根因点 `hibiki_focus_controller.dart` 的 `_geometricTarget` samePane 计算。
- **[x] ① 已修复** — `714b1af45`：把「面板」身份从「最近 Scrollable」升级为「**最近 FocusTraversalGroup**，同组内再按**非空 Scrollable** 细分」（消除「无 Scrollable 的 chrome 与无 Scrollable 的 rail 被判同面板」这个错误等价，非补丁）。home 外壳本就把 rail / body / 设置各包进**独立** FocusTraversalGroup（`home_page.dart:341,370,382`），而宽屏设置 list-detail 是**单个** FTG 包两条独立 ListView——两条信号组合后：rail 与 body chrome 异组→异面板（Down 不再进 rail）、设置两栏同组但异 Scrollable→异面板（保持 BUG-015 / pane-locality 行为）。新增 `_isSamePane()`（① 异 FTG 即异面板；② 同组内两者都在非空且不同 Scrollable 才异面板；任一方无 Scrollable 则只看组）+ `_nearestTraversalGroup()`（`visitAncestorElements` 取最近 FTG 的 Element，按 Element 标识比较，单次 `move()` 内稳定）。排序档次与边缘滚动判定（仍按 Scrollable 的 `activeIsOnlyFocusableInNearestScrollable`）均不动。`flutter analyze` 0 issue、`dart format` 0 改动。opus 子代理审查 🟢 Ready（逐条验证三不变式不破、null-waiver 不误并面板、无 FTG 页降级安全）。
- **[x] ② 已加自动化测试** — `hibiki/test/focus/focus_nav_pane_not_entered_test.dart`：左 rail 与右 body 各一个 `FocusTraversalGroup`，右 body 顶部放无 Scrollable 的页头按钮、下方 ListView 内容；① Down from header 必须落到 `content-*`（修复前落到 `nav-*`，确为红→绿的判别用例）；② Left from content 仍逃到 `nav-*`（跨面板逃逸不被破）。既有不变式守卫 `test/focus/focus_{geometry,pane_locality,left_escapes_pane}_test.dart` 全绿。`flutter test test/focus test/shortcuts test/widgets test/settings` 全量 **476 绿**无回归。
