## BUG-226 · 桌面 hover 视频时字幕被避让顶飞「消失」（底部+右侧列表都不见）
- **报告**：2026-06-12（用户：鼠标放到字幕列表或字幕上面会看不见——①没开字幕模糊②底部当前句字幕 + 右侧字幕跳转列表都会消失③播放中④桌面鼠标 hover / TODO-161）
- **真实性**：✅ 真 bug — 根因 `hibiki/lib/src/media/video/video_subtitle_overlay.dart:427-428`（旧 `_paddingFor` 底部锚点返回 `bottom: widget.bottomPadding + extraBottom`，extraBottom 由 `:449-450` 在控制条可见时取 `controlsBottomReserve`）。
  - 字幕用自绘 `VideoSubtitleOverlay`，TODO-129 把控制条避让改成「控制条可见时在用户基线之上**加** `kVideoControlsBottomReserve`」。基线默认 75（`video_subtitle_style.dart:76`，TODO-129 从 BUG-180 的 100 退回）< 控制条总高 98。
  - 桌面 hover 整个视频区 → `_videoControlsHoverWrap` 的 `MouseRegion.onEnter/onHover`（`video_hibiki_page.dart:4107-4114`）→ `_markControlsVisible(true)` → `_videoControlsVisible=true` → overlay 底部 padding = `75 + 98 = 173px`。字幕 overlay 容器 = `Expanded(Video)` 区（`video_hibiki_page.dart:4088`/`4172`），173px 把字幕**凭空多抬一个基线（75px）**顶进画面中上部 / 顶出可视底带 → 用户看到「字幕消失」。
  - 右侧字幕跳转列表「消失」：鼠标移到 panel（Video 之外的 Row 兄弟，`:4234`）→ 外层 hover wrap `onExit` → `_onVideoControlsHoverExit()`（`:1505-1510`）翻 `_videoControlsVisible=false` → 字幕从 173 抖回 75（旧加法下落差 98px 像闪烁消失）。
  - **纠偏**：`BoxFit.cover`（`:4051`）只裁 libmpv 画面纹理，**不裁** overlay 层（注释 `:4048` 明确）。字幕飞走纯粹是 173px 病态 padding，不是 cover 裁切。病根是避让代数：避让的真实语义是「字幕底缘 ≥ 控制条顶缘」=`max(bottomPadding, reserve)`，被错写成 `bottomPadding + reserve` 加法。
- **[x] ① 根因修复** — 把 `_paddingFor` 底部避让从加法改成对控制条高度取下限：`_paddingFor(anchor, bool controlsVisible)`，底部锚点返回 `bottom: controlsVisible ? max(bottomPadding, controlsBottomReserve) : bottomPadding`（`video_subtitle_overlay.dart:424-443`），`_anchoredPadded` 改传可见性布尔（删 extraBottom 数值叠加，`:445-463`）。控制条隐藏 → 字幕贴 bottomPadding 基线（保 TODO-129 落回贴底优点）；可见 → `max(75,98)=98` 恰骑控制条顶（避进度条又不飞）；手选高位（200）→`max(200,98)=200` 尊重；手选低位（20）→ 可见时抬到 98 躲进度条、隐藏回 20。同步更新 widget 字段注释（`:97-110`）。提交：（见本轮提交哈希）。
- **[x] ② 自动化测试** —
  - `hibiki/test/media/video/video_subtitle_overlay_test.dart`：避让组 5 条几何断言改 `max` 语义——可见时底缘 gap=`reserve`（98，非 75+98=173）；隐藏落回 75；落差=`reserve-75`=23（非整段 98）；新增「manual high(200) 取基线不被改写」「manual low(20) 可见取下限躲进度条」两条。撤回成加法 → `controls visible` 那条产出 173 > 98 红。
  - `hibiki/test/pages/video_subtitle_push_up_guard_test.dart`：新增源码守卫——`_paddingFor` 含 `widget.bottomPadding > widget.controlsBottomReserve`（取下限三元）、不含 `bottomPadding + `（加法）、不残留 `extraBottom`。撤回成加法即露。
