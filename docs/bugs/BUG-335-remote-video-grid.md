## BUG-335 · 手机远端视频显示成横条应改网格
- **报告**：2026-06-19（用户：）
- **真实性**：✅ 真 bug。根因 `hibiki/lib/src/pages/implementations/home_video_page.dart:948-963`（旧实现）——远端（互通）视频区用固定高 200 的横向 `ListView.separated`（`scrollDirection: Axis.horizontal`）+ 每张卡 `SizedBox(width: 260)` 渲染成「一横条横向滚动」，而本地视频区 `_buildGrid`（同文件 `:1190`）用响应式网格 `GridView` + `SliverGridDelegateWithMaxCrossAxisExtent`。两者布局不一致，手机窄屏下远端视频被挤成一横条而非和本地一样的网格卡片。附带：远端 section 标题行 `Row`（`:919`，图标 + 「互通」标题 + 「已配对设备」副标题）两段裸 `Text` 在窄屏（手机）下宽度之和撑爆 Row 右侧溢出。
- **[x] ① 已修复** — 远端视频区改用与本地 `_buildGrid` 完全一致的 `GridView.builder` + `SliverGridDelegateWithMaxCrossAxisExtent(maxCrossAxisExtent: 280, mainAxisExtent: 200, crossAxisSpacing: 12, mainAxisSpacing: 12)`（`shrinkWrap: true` + `NeverScrollableScrollPhysics`，嵌在非 Expanded 的 Column 里随行数自然撑高）；`_buildRemoteVideoCard` 去掉 `SizedBox(width: 260)` 让卡片填满网格 cell（与本地 `_buildCard` 一致）；标题行两段文字各包 `Flexible` + `maxLines: 1` + `TextOverflow.ellipsis` 防窄屏溢出。`hibiki/lib/src/pages/implementations/home_video_page.dart`。提交 1f1967033。
- **[x] ② 已加自动化测试** — `hibiki/test/pages/home_video_remote_grid_layout_test.dart`：widget 行为（极窄手机 240×900 下远端卡片在 `GridView` 祖先内、其祖先链无横向滚动 `ListView`、第二张卡换行排在第一张下方即单列网格）+ 源码守卫（远端 section 复用 `SliverGridDelegateWithMaxCrossAxisExtent`、不再含 `scrollDirection: Axis.horizontal`、不再含固定 `width: 260`）。撤源码修复（还原 develop 横向 ListView 版本）两测试均转红。提交 1f1967033。
- **[x] ③ 复核退回的溢出回归已修复** — 网格化（1f1967033）把远端 `ListView` 换成 `shrinkWrap: true` 的 `GridView` 后，远端 section 放在 `_buildVideoLibraryBody` 的 `Column(children:[remote, Expanded(child:local)])` 的**非 Expanded 槽位**，高度随视频数量无界撑高；远端视频多时整条 `DesktopContentLayout → Column → Expanded → Column` 链无垂直滚动容器 → RenderFlex 溢出（12 视频 360×640 实测溢 **793px**），且把 `Expanded(本地网格)` 挤到高度 0。旧横向 ListView 实现远端有 `SizedBox(height: 200)` 限高故不挤本地，新网格丢了这层限界。根因修复（限高内滚）：`_buildVideoLibraryBody` 用 `LayoutBuilder` 拿本体可用高度，给远端套 `ConstrainedBox(maxHeight: 可用高度 * _kRemoteSectionMaxHeightFraction=0.45)`，本地 `Expanded` 拿剩余 ≥55% 始终 >0；远端 `GridView` 去 `shrinkWrap`/`NeverScrollable` 放进 `Expanded` 占满有界高度并自带垂直滚动，超出卡片靠内滚懒加载消化。不改网格 delegate 参数，窄屏单列网格化不变。守卫 `hibiki/test/pages/home_video_remote_grid_layout_test.dart` 新增：widget（12 远端视频 360×640 无 RenderFlex 溢出 + 本地视频区高度 >0）+ 源码守卫（`_buildRemoteVideoSection` 方法体内无 `shrinkWrap`/`NeverScrollable`，section 用 `LayoutBuilder` + `ConstrainedBox` 按 `_kRemoteSectionMaxHeightFraction` 限高）。撤修复实测两守卫转红（溢 793px），原网格化守卫保持绿。提交 0e9a7e8a3。
