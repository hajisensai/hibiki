- **真实性**：✅ 真 bug。视频页内部已经有字幕跳转列表和行内收藏按钮，但视频收藏句原先只按 `text + bookUid + sectionIndex=null + normCharOffset=null` 存入 `FavoriteSentenceRepository`，没有把 cue 的 `startMs` 写进收藏记录；收藏页又把 `source == video` 的句子当作“不能导航”的特殊行，只能复制/删除，不能回到 `VideoHibikiPage`。因此用户在收藏夹能看到视频句子，却无法识别它对应哪条字幕时间轴，也不能跳回视频并 seek。根因：`hibiki/lib/src/pages/implementations/video_hibiki_page.dart:1688` 一带旧视频收藏缺 cue 锚点；`hibiki/lib/src/pages/implementations/collections_page.dart:243` 一带旧逻辑没有视频打开/seek 分支。
