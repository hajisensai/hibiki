## BUG-061 · 「从本句播放」音频先跳到音频开头→章节开头→才到正确位置（三段跳）
- **报告**：2026-06-05（用户：跳转到 ~100000 字符，用划词弹窗「从本句播放」会先回音频开头，再回章节开头，最后才到正确位置）。采番注：与同轮 A/C 连号取 061（编号可能被并发 agent 抢，以提交为准）。
- **真实性**：✅ **真 bug（显式 seek 加载期瞬态 tick 驱动跨章/reveal）**。沿真实代码路径定位（含 opus 子代理时序追踪）：`audiobook_controller.dart::skipToCue`（`:579`）在 `preload:false`（`load()` 用 lazy 预载）下做跨文件 `seek(index:)`，目标音频文件异步加载期间 `createPositionStream`（125ms tick，`:756`→`_updateCurrentCue`）先吐**瞬态位置**（0 / 旧文件章首 cue.startMs）；每个瞬态 tick 走 `_updateCurrentCue`→`_maybeEmitCrossChapter`（`:820`）→reader `_onCueChanged` reveal，于是「音频开头(0) → 章节开头(旧文件章首) → 正确句(目标加载完成)」三段跳。`skipToCue` 还在 seek 后主动 `_chapterTransition=false`（`:595`），把唯一能挡瞬态的守卫也拆了。被破坏的不变量：**一次显式 seek 期间，positionStream 的瞬态 tick 不得驱动 cue 推进 / 跨章 / reveal / 位置保存**。
- **[x] ① 已修复** — 根因修（在 seek 收口处立「显式 seek 抑制窗」，而非给每个跳变打补丁）：新增 `_explicitSeekInFlight`/`_explicitSeekTargetFileIndex`/`_explicitSeekTargetMs` 三字段 + `_beginExplicitSeek()`，`skipToCue`/`playCueOnce` 在各自 `seek` 前立旗并权威写入目标 cue（sasayaki 路径保持立旗，非 sasayaki `idx<0` 路径维持原「按当前位置即时定位」语义不抑制）。`_updateCurrentCue` 顶部据纯谓词 `reachedExplicitSeekTargetForTesting`（player 已切到目标音频文件 **且** 位置到达 `targetMs-300ms` 容差）判定落定：未落定 → `return` 抑制瞬态 tick；落定 → 清旗后正常处理。hibiki_audio 包内 `dart analyze` 0 issue。
- **[x] ② 已加自动化测试** — `hibiki/test/media/audiobook/skip_to_cue_transient_seek_test.dart`：① 纯谓词 4 例（index 未切 / 位置仍 0 → 抑制；到达 target-容差 / 越过 target → 放行）red→green；② 源码扫描守卫 1 例（钉住 `skipToCue`+`playCueOnce` 都 `_beginExplicitSeek`、`_updateCurrentCue` 顶部有 `if (_explicitSeekInFlight)` guard 且走谓词，防接线被回归删）。`test/media/audiobook/` 全量 282 例绿（含 `audiobook_controller_seek_test`/`audiobook_position_flush_test` 等 player 驱动用例无回归）。
- **备注**：有声书播放/seek 时序类。纯谓词 + 现有 player 驱动回归覆盖；驱动 fake just_audio positionStream 模拟完整三段跳序列成本高且非本仓范式，故未加。**真机肉眼复测原始失败路径待用户**：跳到远处后「从本句播放」应一步到位，无三段跳。
- **审查修正（opus，W-2）**：`_explicitSeekInFlight` 是跨 tick 的旗，暂停态 skipToCue 后无 tick 会滞留 true，若期间发生经 `_updateCurrentCue` 重算的边界调用（`setChapterCues`/`notifySectionRestoreCompleted`）会被错误抑制。已在这两个上下文边界显式复位 `_explicitSeekInFlight = false`（播放态首 tick 落定本就自愈，边界复位补齐暂停态/跨章泄漏面）。
