## BUG-084 · 本机作为 Hibiki 服务端时点「立即同步」误报「请先设置同步后端」
- **报告**：2026-06-06（用户：「本机作为 hibiki 服务器的时候，点立即同步会说请先设置同步后端」）。采番注：本 worktree 基线最大号 BUG-083（本轮 BUG-083 已占），develop 上并发已占 076（m3u8 拖入），故取 077（仍可能被并发抢，以提交为准）。
- **真实性**：✅ **真 bug（UX 误导，沿真实代码路径定位）**。「作为服务端」(`isServerEnabled`，repo key `sync_server_enabled`) 与「出站后端类型」(`getBackendType`/`SyncBackendType`) 是**两个独立维度**：启用服务端从不设置出站 backend（唯一调 `setBackendType(hibikiServer)` 的地方是「连出去当客户端」配对，`sync_settings_schema.dart:2386`）。于是纯服务端机器 `getBackendType()` 落到默认值 `googleDrive`（未登录），`runManualFullSync`（`sync_auto_trigger.dart:200-206`）→ `resolveSyncBackend(googleDrive)` → `restoreAuth` → `isAuthenticated==false` → 返回 `notConfigured` → `_SyncNowWidget` 显示 `t.sync_compare_unavailable`「请先设置同步后端」。但按设计**纯服务端是被动数据源**（由客户端拉/推，自身无出站同步语义，见 `sync_settings_schema.dart:107-111` 与 `hibiki_sync_server.dart` 类注释——服务端只跑 WebDAV 请求处理，从不 `resolveSyncBackend`/跑 orchestrator）。所以"立即同步"在服务端模式下根本不该出现，提示是误导。根因 `hibiki/lib/src/sync/sync_settings_schema.dart`（手动同步区不随 `serverEnabled` 门控）。
- **[x] ① 已修复** — 本提交（自含修复+测试+本条目）。**根因修（按角色门控 UI，非补丁）**：手动同步区（Group 4）的 `sync.sync_now` / `sync.compare` 两项加 `visible: (ctx)=>!serverEnabled`，并新增 `sync.server_mode_note` 信息行（`visible: serverEnabled`）显示 `sync_server_mode_active`「本机作为同步服务端」+ `sync_server_mode_clients_drive`「由已连接的客户端发起同步，本机无需手动同步」；`_SyncSettingsState.setServerEnabled` 末尾补 `_settingsContext.refresh()` 让切换服务端开关时手动同步区**实时**重新门控（不只下次进页面）。账户区（`sync.account_status`）本就 `visible: isOAuthSyncBackend(backendType)`，服务端模式下 backendType=hibikiServer 已天然隐藏，无需改。**修正（同轮 followup）**：门控最初只判 `serverEnabled`，但该 flag 在切到云后端后仍持久存在（真机 DB 实测 `sync_server_enabled=true` 同时 `backendType=googleDrive`），会误把云后端的「立即同步」也隐藏。改为 `_isHostingInterconnect(ctx)` = `serverEnabled && backendType==hibikiServer`（两者与门），仅互联 host 角色才隐藏。
- **[x] ② 已加自动化测试** — `hibiki/test/sync/sync_settings_visibility_test.dart`（行为 + 源码守卫）：① 更新 section[3] 期望 ids 为 `[server_mode_note, sync_now, compare]`；② 断言三项各带 `visible` 谓词（非无条件）；③ 源码守卫断言三个 `visible:` 都引用 `serverEnabled`（防被改门控到错误条件）。i18n 经 `tool/i18n_sync.dart --add` 加 2 key（17 语言完整）+ `dart run slang` 重生成。`test/sync/` + `test/settings/` + `test/i18n/` 全量 410 绿。
- **备注**：sync/UI 门控类。代码正确 + 无回归；**Windows 桌面真机复测**（开启服务端 → 进「同步与备份」→ 看不到"立即同步"、改显示服务端说明行）**待用户**。与 [[BUG-085]]（服务端切出界面被关）同属服务端模式修复一轮。
