How to use this

Each board has A, B, and C directions. Use the pick buttons under each board, or mix details like "04 B with A bottom bar". Board 17 is documentation-only; pick it only if you prefer a different map layout.

The current design package covers 3 entry surfaces, 53 page surfaces, and 28 shared/support UI surfaces. Runtime Flutter code is intentionally untouched until the choices are locked.

If the full board image is too dense, open the variant gallery to inspect each A/B/C mockup as its own crop. If you want every exact interface listed with three images, open the interface gallery.
