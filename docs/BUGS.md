## BUG-035 · Hibiki 互联：主机可达却报「No reachable Hibiki server address」（新开启服务器从未创建 WebDAV 根目录）
- **报告**：2026-06-04（用户：「No reacable Hibiki server address」。澄清确认：场景 A —— **主机服务器确认开着且可达**（浏览器/ping 能连），客户端仍报不可达 = 真 bug）。
